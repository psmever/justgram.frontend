# Justgram.Frontend

## psmever's Justgram Fornt-End Source.

#### Config
```
cd ${project Root}
cp config/sample.environment.env setting/local.env
```

#### Node Module Install.
```
yarn install
```

### Local Develper
```
yarn start
```

### Build
```
yarn build
```
